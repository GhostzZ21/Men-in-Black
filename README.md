# Men-in-Black
Proyecto de la universidad (Y), el que lo robe es gay.

Diseñado para expandir el contenido semántico del sitio web mediante una sección dedicada a la filosofía de la startup (etiqueta `<section id="manifiesto">`). El prompt instruyó a la IA a organizar el bloque de texto en columnas fluidas aplicando propiedades de Flexbox (`flex-wrap: wrap`), asegurando que la tipografía mantuviera una alta legibilidad sobre fondos oscuros y destacando conceptos clave con el color de acento corporativo.
